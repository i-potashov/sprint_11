### Introduction
The project includes the ability to customize the user data of the page and add image cards, followed by the ability to like and delete the card added by the user.

**Start project:**
```sh
$ npm install
```
**Run build:**
```sh
$ npm run build
```
**Config develop:**
```sh
$ npm run dev
```
**Load to gh-pages:**
```sh
$ npm run deploy
```
